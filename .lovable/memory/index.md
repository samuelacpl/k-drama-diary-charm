# Project Memory

## Core
React, Tailwind CSS. Cloud sync via Lovable Cloud + localStorage fallback.
Aesthetic: Cute, minimal pastel palette (pink, lavender, beige), rounded UI.
Fonts: Quicksand and Playfair Display.
UI Rule: Watch status must be positioned above episode tracking.
When a drama is 'Completed', auto-sync watched episodes to total count.
Auth: Email/Password + Google OAuth. All routes protected except /auth.
